# forthepiece

1인칭 3D 좀비 VR FPS게임.

Unity를 활용한 Oculus VR게임.

제작기간: 2020.11.04~ 2020.12.04



## 제작: 팀 Soldier
 
  팀장: 임영택
  
  팀원: 최재혁, 손민정, 이승현


# 주의사항

  VR Oculus 환경 기준으로 제작된 게임이므로 다른 환경에선 플레이가 제대로 되지 않을 수도 있습니다.

  화면 전환이 매끄럽지 않고 최적화가 이루어져있지 않으므로 심한 3D 멀미를 유발할 수 있습니다.(특히 회전할 때 멀미유발이 심합니다)
  

## 조작법(Oculus)

 왼쪽스틱: 상하좌우 이동

 왼쪽 Y: 아이템 획득

 왼쪽 X: UI tab
 
 왼쪽 grip: 점프
 
 왼쪽 trigger: 대쉬
 
 오른쪽스틱: 회전
 
 오른쪽 trigger: 총발사
 
 오른쪽 grip: 무기 전환
 
 오른쪽 B: 상호작용
